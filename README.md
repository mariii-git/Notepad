Notepad Application (C# WinForms)
A simple Windows Forms Notepad application built with C# that provides basic text editing features similar to the classic Windows Notepad.
Features
· Create, open, edit, and save text files
· Cut, copy, paste, undo, and select all text operations
· Print text documents using Windows printing services
· Clean and simple WinForms user interface
· Uses RichTextBox for text editing
Technologies
· C#
· .NET Framework (Windows Forms)
· File I/O (System.IO)
· Windows Forms UI
Usage
Run the application
Type or edit text in the editor area
Use the menu options to open, save, or print text files
Perform basic text editing operations (copy, paste, cut, undo)
